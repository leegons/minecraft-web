# 🌍 方块世界 (Minecraft Web)

[English Version](README_en.md) | [中文版本](README.md)

一个基于 **Three.js** 开发的高性能网页版体素沙盒游戏。本项目完全由代码编写，支持地形生成、建筑、昼夜循环、存档等核心功能，且不依赖外部资源包。

## ✨ 功能特性

- 🌲 **动态地形生成**：使用噪声算法生成高山、平原、湖泊和森林。
- 🌓 **动态环境系统**：完整的昼夜交替，包含太阳、月亮运行轨迹，星空及动态变化的云层。
- 🧱 **构建与挖掘**：支持生存（资源有限）和创造（资源无限）两种基本游戏模式。
- 💾 **自动存档**：游戏状态实时通过 `localStorage` 自动保存。
- 🚀 **性能优化**：视锥剔除、面剔除、区块动态加载及实例化渲染，确保在浏览器中流畅运行。
- 📱 **多端适配**：支持 PC 的键盘鼠标以及移动终端的虚拟摇杆。
- 🎵 **音效生成**：通过 Web Audio API 程序化实时合成音效，无需加载外部音频文件。

## 🕹️ 操作指南

### ⌨️ 桌面端 (PC)
- **W, A, S, D**: 移动
- **空格**: 跳跃
- **双击空格**: 在创造模式下切换/飞行
- **鼠标左键**: 挖掘
- **鼠标右键**: 放置方块
- **数字键 1-9**: 切换方块类型
- **F3**: 开启/关闭调试信息面板

### 📱 移动端
- **左侧摇杆**: 移动
- **右侧区域**: 滑动转头
- **右下按钮**: 跳跃、挖掘、放置

## 🛠️ 安装与运行

确保您的计算机已安装 [Node.js](https://nodejs.org/)。

1. **克隆仓库**
   ```bash
   git clone https://github.com/leegons/minecraft-web.git
   cd minecraft-web
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **运行开发服务器**
   ```bash
   npm run dev
   ```

4. **构建生产版本**
   ```bash
   npm run build
   ```

## 📂 项目结构说明

- `src/world.ts`: 负责世界的核心逻辑与网格构建。
- `src/player.ts`: 玩家物理模拟与控制。
- `src/sky.ts`: 天空环境渲染。
- `src/textures.ts`: 纹理生成逻辑。
- `src/interaction.ts`: 射线检测交互。

详细的项目结构请查看 `artifacts/code_structure.md`。
